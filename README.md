# Network Labs

Repositório para documentar e organizar laboratórios de Redes de Computadores, desenvolvidos principalmente utilizando o **Cisco Packet Tracer**. 

O objetivo é consolidar e compartilhar conhecimentos práticos em topologias de rede, roteamento, comutação (switching), serviços de rede entre outros.

---

## Principais Tecnologias e Ferramentas Utilizadas
* **Cisco Packet Tracer** (Criação e simulação das topologias)
* **Cisco IOS** (Configuração via CLI)
* **Git & GitHub** (Versionamento e documentação)

## Como utilizar este repositório

Para replicar qualquer laboratório no seu Packet Tracer local:
1. Navegue até a pasta do laboratório desejado.
2. Abra a imagem `topologia.png` para entender a estrutura física.
3. Crie os dispositivos no seu Packet Tracer e utilize os arquivos `.cfg` (como `R1_router.cfg`) para aplicar as configurações via CLI (Digitando ou colando os comandos).
